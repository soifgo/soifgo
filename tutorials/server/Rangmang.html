RANGMANG Dance Mode Configuration

This configuration defines how the RANGMANG module controls LED dance modes. Each row encodes the following parameters:

  • Channel – 2 digits identifying the channel.
  • Mode – 2 digits (01–38). Each mode represents a dance style.
  • Red / Green / Blue – 3 digits each. Normal range is 0–250. Special values:
    • 251 → Random selection from lower range
    • 252 → Random slightly above 250
    • 253 → Random mid-range
    • 254 → Random higher than mid-range
    • 255 → Random maximum
  • Dance Time 1 – 3 digits, milliseconds per pixel move (first timing).
  • Dance Time 2 – 3 digits, milliseconds per pixel move (second timing).
  • Pixel Count – 3 digits, number of dancing pixels.
Channel Mode Red Green Blue Dance Time 1 (ms) Dance Time 2 (ms) Pixel Count
01 12 251 251 251 001 001 128

RANGMANG Fixed Color Mode Configuration

In fixed color mode, the configuration format is slightly different:

  • Channel – 2 digits (e.g., 01 for module 1).
  • Mode – must always be 00.
  • Red / Green / Blue – 3 digits each, range 000–255.
  • Start Pixel – 3 digits, the first pixel to color.
  • End Pixel – 3 digits, the last pixel to color.
  • Memory Slot – 3 digits, memory index (001–100 available).
Channel Mode Red Green Blue Start Pixel End Pixel Memory Slot
01 00 123 123 123 001 045 001

Power Regulation Notes

  • For 5V LED strips: You can remove the regulator and directly power the microcontroller from the strip's 5V socket.
  • For 12V LED strips: You must use a 12V-to-5V regulator to power the microcontroller safely.
  • Note: To prevent reverse connection of the LED strip to the module, use a properly keyed connector. If necessary, add a diode in series with the microcontroller's power input to protect against polarity reversal.
