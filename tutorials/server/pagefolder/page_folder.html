Navigation in SoifGo

In SoifGo you can create multiple branches such as folders and subfolders. Each folder may contain separate pages, and these pages remain independent.

Page Opening Behavior

With the page-opening behavior, you can assign a button to load another page you have designed. From that page, you can also navigate back to the previous one.

Folder Opening Behavior

Sometimes you need to access pages located in another folder. In this case, simply choose the folder-opening behavior and select the desired folder.
