SoifGo

SoifGo is the natural evolution of the SoifDesign project—focused on usability, interaction, and respect for the user.

This Android application offers advanced Bluetooth communication, RGB control, file management, and sensor-based interactions, all designed to deliver a real, tested experience for users.

All documentation is crafted with clarity, precision, and print-friendly formatting to ensure no user is left confused or frustrated.

Features

  • Serial Bluetooth communication with customizable formats
  • RGB lighting control for WS2812, WS2813, WS2815, and WS2811 modules
  • File management for HTML, PDF, images, and notes
  • API and server interaction via GET/POST requests
  • Sensor-based effects using phone's light, motion, and magnetic sensors
  • Graph plotting with zoom, pause, and resume capabilities
