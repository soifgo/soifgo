README

SoifGo is the natural evolution of SoifDesign—focused on usability, interaction, and respect for the user.

  • Bluetooth serial communication with customizable formats
  • RGB control for WS281x modules
  • File management for HTML, PDF, images, and notes
  • Sensor-based effects using phone sensors
  • Graph plotting with zoom, pause, and resume

For full documentation, visit the GitHub repository.
