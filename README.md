## 📱 About SoifGo
SoifGo is an Android application that provides users with advanced capabilities using **buttons and seek bars**.  

### 🔧 Features
- 📡 **Bluetooth communication:** Sending and receiving serial Bluetooth data  
- 🌐 **API server interaction & web browsing**  
- 📝 **Managing HTML files & notes**  
- 📂 **Shortcut & library management for PDFs and images**  
- 🎛️ **Seek bar control with serial Bluetooth transmission**  
- 🎨 **Customizable buttons (size, color, shape, vibration, behavior)**  
- 🔄 **Dynamic effects based on Bluetooth input**  
- 📊 **Graph storage & review inside buttons**  
- 🎚️ **Predefined seek bar values for enhanced interaction**  
- 🌈 **Optimized RGB color control (WS2812, WS2813, WS2815, WS2811)**  
- ⚙️ **Bluetooth format selection (CR+LF, CR, LF, NULL, …)**  
- 🔁 **Smart Bluetooth resend (Check-in)**
